# Print group sequential CRT design

Print group sequential CRT design

## Usage

``` r
gsPrintCRT(x)
```

## Arguments

- x:

  Object of class `gsDesignCRT` created by
  [`gsDesignCRT()`](https://leejtding.github.io/gsDesignCRT/reference/gsDesignCRT.md).

## Value

Printed output of design parameters and table of stopping boundaries
